Refonte CRM - Vague 3 - Golden v1

Signaux condensés, Prospection campagne, Dashboard temporel

Les 3 chantiers de la Vague 3, dans le système premium déjà validé en Vague 2 (mêmes tokens, mêmes primitives). À valider dans Chrome avant toute ligne de code production. Garde-fous respectés : recherche visible (jamais Cmd+K), bornage 1200px, couleur porteuse de sens (palette workflow), états vides soignés.

Décisions tranchées avec toi : Signaux = condenser le haut (2 bandes) + action Archiver (réversible) regroupée dans un menu ... ; Prospection = colonne Campagne (étiquette posée à l'import, optionnelle, cumulative) + filtre par campagne + export CSV (déjà livré) ; Dashboard = accueil organisé par urgence dans le temps (en retard / aujourd'hui / cette semaine) + fil d'activité, plutôt qu'un mur de chiffres.
