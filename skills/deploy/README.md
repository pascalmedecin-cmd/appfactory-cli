# Skill Deploy

Push -> Vercel preview/prod. Tests automatises. Configuration domaine.

## Responsabilites
- Lancer les tests (Vitest + Playwright)
- Push vers GitHub
- Deployer sur Vercel (preview ou production)
- Configurer le domaine personnalise (production)
- Reporter le statut a l'operateur

## Input
- Projet SvelteKit genere et valide
- Cible : preview ou production

## Output
- URL de preview ou URL de production
- Rapport de tests
- Statut de deploiement
