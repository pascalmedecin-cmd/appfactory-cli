Règles & anti-patterns

Règle d'or : aucun pattern d'un archétype copié dans un autre. Si tentation = signal d'un mauvais mapping → revérifier le mapping.

✓ Must have

  • H1 sémantique unique par page
  • ARIA tabs underline si > 1 scope
  • DataTable table-fixed + sticky 2 cols left
  • ScorePill systémique sur tout score visible
  • aria-label sur tr cliquable
  • Empty state contextualisé par filtre
  • ImportModal 3 parcours pour tout import
  • ConfirmModal pour toute action destructive
  • Focus-visible global (outline 2px primary)
  • Tabular nums sur compteurs / numéros

✗ Do NOT

  • Pas de hero éditorial sur page workspace ou consommatrice non-veille
  • Pas de DataTable workspace dans editorial
  • Pas de mag-* typography hors module /veille
  • Pas de fond primary-light en body workspace
  • Pas de funnel décoratif, pas de progress steps imaginaires
  • Pas de Badge Chaud/Tiède/Froid → ScorePill
  • Pas de window.confirm() → ConfirmModal
  • Pas de filtrage client-side > 50 lignes
  • Pas de select déguisé en tabs
  • Pas d'option « Autre » dans les select
  • Pas de surcharge tokens locale (tout passe par tokens.json)
